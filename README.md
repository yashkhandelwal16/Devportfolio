# 💼 DevPortfolio

A modern, responsive developer portfolio built to showcase my projects, skills, achievements, and experience with a clean and interactive UI.

## 🚀 Features

- 📱 Fully Responsive Design
- 🎨 Modern & Clean UI
- 🌙 Dynamic Theme Support
- ✨ Smooth Animations
- 📂 Projects Showcase
- 📜 Certificates & Achievements
- 📞 Contact Section

## 🛠️ Tech Stack

- **Frontend:** React.js
- **Styling:** CSS / SCSS
- **Icons:** Font Awesome, Devicon
- **Animations:** React Reveal, Lottie React
- **Deployment:** Vercel

## ⚙️ Installation

```bash
git clone https://github.com/yashkhandelwal16/Devportfolio.git
cd Devportfolio
npm install
npm start
```

Visit: `http://localhost:3000`
